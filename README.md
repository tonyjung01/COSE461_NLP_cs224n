# COSE461 Natural Language Processing

This repository contains coursework for COSE461 Natural Language Processing at Korea University, based on Stanford CS224N, taught by Professor Byungjoon Lee.

## Assignment1

Assignment1 focuses on exploring word vectors. It covers count-based word vectors, co-occurrence-based representations, and pretrained embeddings to build intuition about how word meaning can be represented in vector space.

## Assignment2

Assignment2 is a neural dependency parsing project implemented in PyTorch. It focuses on transition-based parsing, parser state representations, and training a neural parser for syntactic dependency prediction.

## Assignment3

Assignment3 is a Neural Machine Translation project implemented in PyTorch. It focuses on building a sequence-to-sequence NMT model with attention, including training, decoding, and evaluation with BLEU.
